# AWS RDS PostgreSQL ETL & SQL Analytics Project

## Project Overview

This project demonstrates an end-to-end ETL pipeline using:

- AWS S3
- Python
- PostgreSQL AWS RDS
- SQL Analytics

The project performs:

- CSV data extraction from AWS S3
- Data loading into PostgreSQL RDS
- CRUD operations
- SQL analytics queries
- CSV export from RDS
